## 設定  
Azureポータルの自分が作ったアプリの「設定」の「アプリケーション設定」一番下の「仮想アプリケーションとディレクトリ」で  
「仮想ディレクトリ」 /post   
「サウトルートを基準とした物理パス」 site\wwwroot  
「アプリケーション」 にチェック  
で可能
